# Retail Sales Performance Dashboard (Power BI)

## Project Overview
This project analyzes retail sales performance using an interactive Power BI dashboard built from the Superstore dataset.  
The goal of the dashboard is to provide insights into company revenue, profitability, regional performance, and sales trends.

The dashboard highlights key business metrics and allows users to explore performance across regions, product categories, and time.

## Key Metrics
The dashboard tracks three key performance indicators:

- **Total Sales:** $2.30M  
- **Total Profit:** $286.32K  
- **Profit Margin:** 12.47%

These KPIs provide a quick overview of overall business performance.

---

## Key Insights

### Regional Performance
Sales performance varies across regions, with the **West region generating the highest revenue**, while the **South region shows the lowest sales**.

### Category Profitability
The **Technology category generates the highest profit**, followed by Office Supplies, while Furniture contributes significantly less profit.

### Product-Level Profitability
Sub-category analysis reveals that certain products produce negative profits, highlighting opportunities for pricing or cost adjustments.

### Sales Trends
Monthly sales analysis shows increasing sales toward the end of the year, suggesting seasonal demand patterns.
